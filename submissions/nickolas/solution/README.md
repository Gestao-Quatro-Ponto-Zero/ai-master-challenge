## Uso dos dados reais

A solução utiliza diretamente os datasets fornecidos:

- sales_pipeline.csv → oportunidades
- accounts.csv → dados da empresa

A lógica de scoring foi construída a partir de proxies reais:

- ICP → receita da empresa
- dor → valor do deal
- timing → tempo entre engage e close
- estágio → avanço no funil

Isso transforma dados brutos do CRM em priorização acionável.
