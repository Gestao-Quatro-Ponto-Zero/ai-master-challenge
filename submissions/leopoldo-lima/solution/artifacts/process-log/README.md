# Saídas técnicas (`solution/artifacts/process-log/`)

Esta pasta é onde **scripts** da solução gravam evidências quando correm a partir de `solution/` (ex.: `export-real-flow-evidence`, gravação de demo, capturas automatizadas).

**Evidências versionadas para o avaliador** (notas, JSON de fluxo real, guias, imagens entregues) estão na pasta canónica da submissão:

`submissions/leopoldo-lima/process-log/` (irmã de `solution/`).

Após gerar arquivos aqui, copie ou mova o que for relevante para `process-log/` antes do PR, para manter o process log num único sítio legível.
