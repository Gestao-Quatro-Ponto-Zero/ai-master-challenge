# CRP-SUB-01 — Auditoria contra o submission guide

## Objetivo
Repassar o guide oficial e marcar, item por item, o que já está coberto e o que ainda falta.

## Requisitos do guide a validar
- solução funcional
- process log obrigatório
- formatos aceitos de evidência
- submissão via PR
- README baseado no template
- presença de Executive Summary, Abordagem, Resultado, Recomendações e Limitações
- uso estratégico de IA com iteração e julgamento

## Tarefas
1. Criar `docs/SUBMISSION_GUIDE_AUDIT.md`.
2. Montar tabela:
   - requisito
   - status (`atende`, `parcial`, `não atende`)
   - evidência
   - CRP que fecha a lacuna
3. Atualizar `docs/CHALLENGE_CHECKLIST.md` se necessário.
4. Atualizar `LOG.md` e `PROCESS_LOG.md`.

## Entregáveis
- auditoria do guide
- checklist atualizado
- logs atualizados

## Impacto na Submissão
Fecha lacunas formais antes do PR final.

## Evidências obrigatórias
- tabela auditável
- links para arquivos reais
- validação humana da classificação de status

## Definition of Done
- auditoria concluída
- lacunas restantes explicitadas
- próximos CRPs apontados
- logs atualizados
