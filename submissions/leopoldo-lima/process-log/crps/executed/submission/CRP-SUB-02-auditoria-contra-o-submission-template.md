# CRP-SUB-02 — Auditoria contra o submission template

## Objetivo
Garantir aderência literal ao template oficial da submissão.

## Tarefas
1. Criar `docs/SUBMISSION_TEMPLATE_AUDIT.md`.
2. Comparar o README final da submissão com cada bloco do template:
   - Sobre mim
   - Executive Summary
   - Solução
   - Abordagem
   - Resultados / Findings
   - Recomendações
   - Limitações
   - Ferramentas usadas
   - Workflow
   - Onde a IA errou e como corrigi
   - O que eu adicionei que a IA sozinha não faria
   - Evidências
3. Marcar status e corrigir divergências.
4. Atualizar `LOG.md` e `PROCESS_LOG.md`.

## Entregáveis
- auditoria do template
- correções no README final
- logs atualizados

## Impacto na Submissão
Reduz a chance de um README “bom”, mas desalinhado com o template oficial.

## Evidências obrigatórias
- matriz template x README
- diff das correções
- revisão humana final

## Definition of Done
- template 100% coberto
- README final corrigido
- logs atualizados
