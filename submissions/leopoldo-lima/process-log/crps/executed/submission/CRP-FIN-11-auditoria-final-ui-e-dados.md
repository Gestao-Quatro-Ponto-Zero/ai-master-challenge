# CRP-FIN-11 — Auditoria final: UI, dados e coerência do ranking

## Objetivo
Fazer uma revisão final para garantir que UI, dados reais, score e explicações contam a mesma história.

## Fazer
- auditar:
  - KPIs
  - ranking
  - detalhe
  - filtros
  - modo padrão
- verificar se dados exibidos batem com o dataset real
- revisar inconsistências de label, idioma, status e explicação

## Impacto na submissão
Fecha o ciclo de verificação humana e reduz risco de contradições visíveis.

## Evidências obrigatórias
- checklist preenchido
- capturas da home, filtro, ranking e detalhe
- nota das inconsistências encontradas e corrigidas

## Atualizações obrigatórias de process log
Registrar:
- lista de checagens executadas
- inconsistências encontradas
- correções aplicadas

## Definition of Done
- auditoria concluída
- sem inconsistências gritantes entre UI, dados e score
- evidências finais capturadas
