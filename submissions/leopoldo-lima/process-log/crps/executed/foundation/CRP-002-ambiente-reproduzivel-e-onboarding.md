# CRP-002 — Ambiente reproduzível e onboarding

## Objetivo
Permitir que qualquer avaliador rode o projeto sem atrito desnecessário.

## Entregáveis
- `.env.example`
- `Makefile` ou scripts equivalentes
- `docs/SETUP.md`

## Tarefas
1. Levantar variáveis de ambiente
2. Criar comandos simples: install, dev, test, lint, format, typecheck, build
3. Reutilizar scripts existentes quando fizer sentido
4. Atualizar README com Quick Start

## Critérios de aceite
- Setup validado localmente
- Nada de comandos fictícios
- Sem segredos reais no repositório

## DoD
- Setup documentado
- Scripts funcionais
- LOG.md atualizado
