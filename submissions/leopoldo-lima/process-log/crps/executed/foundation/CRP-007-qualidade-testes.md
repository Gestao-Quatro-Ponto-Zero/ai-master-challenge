# CRP-007 — Qualidade: testes de unidade, integração e smoke

## Objetivo
Sair do 'funciona na minha máquina'.

## Entregáveis
- suíte mínima confiável
- cobertura de fluxos críticos
- `docs/TEST_STRATEGY.md`

## Tarefas
1. Mapear fluxos críticos
2. Criar testes unitários para score e utilitários
3. Criar integração para serviços/rotas
4. Criar smoke test da aplicação

- Testes pequenos e confiáveis
- Cobertura útil, não cosmética
- Comando único de execução
