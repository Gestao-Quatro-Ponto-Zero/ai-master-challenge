# CRP-CBX-07 — Polimento visual dos filtros

## Objetivo
Fazer parecer produto, não formulário cru.

## Fazer
- alinhar largura e altura dos campos
- indicar claramente qual filtro é digitável
- reduzir cara de input solto
- manter consistência com o restante da UI
- considerar chips/label do filtro selecionado numa etapa posterior

## Definition of Done
- filtros parecem parte do cockpit
- não há desalinhamento gritante
- visual antes/depois documentado

## Evidência
- screenshot comparativa
- nota curta em `docs/UX_DECISIONS.md`
- validação humana explícita
