# CRP-UX-07 — Transformar filtros em ferramenta, não formulário

## Objetivo
Fazer os filtros ajudarem decisão em vez de parecerem cadastro administrativo.

## Problema
Filtros mal resolvidos passam sensação burocrática e aumentam atrito.

## Tarefas
1. Revisar UX dos filtros:
   - região
   - manager
   - estágio
   - score band
   - busca por conta ou ID
2. Melhorar compactação e legibilidade.
3. Adicionar limpar filtros.
4. Mostrar contagem de resultados.
5. Garantir comportamento consistente com dados reais.

## Impacto na Submissão
- Melhora usabilidade percebida.
- Reforça que a solução é operacional.

## Evidências obrigatórias
- screenshot dos filtros revisados
- evidência de aplicação real dos filtros
- diff da barra/painel de filtros
- nota humana sobre a redução de atrito

## Atualizações obrigatórias de process log
Registrar:
- problemas da UX anterior
- decisões de simplificação
- comportamento dos filtros após ajuste
- evidência funcional

## Atualizações obrigatórias de README/Submission
- atualizar screenshots se a barra/painel mudar significativamente

## Definition of Done
- filtros úteis e legíveis
- contagem de resultados visível
- ação de limpar filtros presente
- estados coerentes com dados reais
- `PROCESS_LOG.md` atualizado
- evidências salvas
- README atualizado quando aplicável
- verificação humana explícita realizada
