## 📌 Executive Summary

The **Churn Intelligence System (CIS)** goes beyond traditional dashboards.

It combines:

- Machine Learning (XGBoost)
- Generative AI (LLM)
- FastAPI backend
- Product-oriented frontend (Lovable)

Goal: transform churn risk into **executive decisions and actionable workflows**

---

## 🧠 Core Value Proposition

CIS connects:

- Prediction (ML)
- Financial impact (MRR at Risk)
- Prioritization (AI)
- Decision (Executives)
- Execution (Kanban)
- Approval (WhatsApp)

👉 From data → to decision → to execution

---

## 🏗️ Architecture (High-Level)

Data → Feature Engineering → ML Model → Risk Scoring → AI Insights → Recommendations → Approval → Execution

- MRR Active

- ARPU
